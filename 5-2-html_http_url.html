5.2. Entstehung von HTML, HTTP und URLs

Tim Berners-Lee erkannte, dass die Verknüpfung von Dokumenten durch Hyperlinks der Schlüssel zu einem effektiven Informationssystem sein würde. Um diese Vision zu verwirklichen, benötigte er jedoch eine standardisierte Methode zur Strukturierung und Darstellung von Dokumenten. 1989 entwickelte er daher die “Hypertext Markup Language” (HTML), eine einfache Auszeichnungssprache, die Text, Bilder, Hyperlinks und andere Elemente in einem Dokument definiert. HTML-Dokumente bestehen aus Textzeilen, die mit HTML-Tags markiert sind. Diese Tags geben dem Browser Anweisungen, wie die Inhalte auf dem Bildschirm darzustellen sind.

Neben HTML benötigte Berners-Lee auch ein Protokoll für die Übertragung von HTML-Dokumenten zwischen Computern. 1989 entwickelte er daher das “Hypertext Transfer Protocol” (HTTP). HTTP ist ein client-serverbasiertes Protokoll, welches definiert wie Anfragen und Antworten zwischen einem Webbrowser (Client) und einem Webserver (Server) ausgetauscht werden. Um die einzelnen Seiten innerhalb des World Wide Webs eindeutig zu identifizieren, führte Berners-Lee außerdem das “Uniform Resource Identifier” (URI)-System ein. Heutzutage ist dieses System auch unter dem Namen “Uniform Resource Locators” (URLs) bekannt. Ein URI besteht aus mehreren Teilen, darunter einem Protokoll (z. B. http), einem Hostnamen bzw. einer IP-Adresse (z. B. 53.109.15.210) und einem Pfad zu dem aufrufenden Element (z. B. /index.html).

HTML, HTTP und URLs bilden die Grundlage des World Wide Web. HTML definiert die Struktur und den Inhalt von Webseiten, HTTP ermöglicht die Übertragung von HTML-Dokumenten zwischen Webbrowsern und Webservern, und URLs dienen als eindeutige Adressen zur Identifizierung von Webseiten und anderen Ressourcen im Internet.

Um seine theoretische Ausarbeitung des World Wide Webs in die Praxis umzusetzen, entwickelte Berners-Lee außerdem den ersten Webserver der Welt, den “CERN httpd”. Dieser rudimentäre Server konnte die Dokumente zwar nur statisch anzeigen, war jedoch ein wichtiger Schritt in der Entwicklung des World Wide Webs.

Auch der erste Browser (Client) kam von Berners-Lee. Er trug den Namen “WorldWideWeb” und lief wie sein Webserver httpd auf dem Betriebssystem “NeXTStep”. Auch dieser Browser war in seinen Funktionen stark limitiert. So wurde beispielsweise das Darstellen von Grafiken noch nicht unterstützt und man beschränkte sich auf Text und Hyperlinks. 1990 nahm Berners-Lee die erste Seite im World Wide Web online. Diese ist auch noch heute unter http://info.cern.ch erreichbar.
