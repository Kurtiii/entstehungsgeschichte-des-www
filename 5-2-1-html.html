5.2.1. HTML - Aufbau und wichtige Tags

Ein HTML-Dokument besteht grundsätzlich aus zwei wesentlichen Teilen. Es startet mit einem sogenannten “Head” (dt. Kopf), welcher Metainformationen über das entsprechende Dokument wie dessen Titel bzw. eine kurze Beschreibung enthält. Die im Head angegebenen Informationen werden nicht im Browser-Fenster für den Benutzer dargestellt. Der zweite Teil ist der sogenannte “Body” (dt. Körper). Er enthält den eigentlichen Inhalt des Dokumentes, also beispielsweise die Texte, Grafiken oder Hyperlinks.

HTML ist eine Tag-basierte Auszeichnungssprache. Es dient lediglich zur Strukturierung der Dokumente und kann selbst keine logischen Aktionen durchführen, wodurch HTML keine Programmiersprache ist. Ein HTML-Tag startet mit einem opening tag z.B. <p>. Anschließend folgt der Inhalt des entsprechenden Tags. Nach dem Inhalt folgt ein schließender closing tag z.B. </p>. Da der p-Tag für paragraph (dt. Absatz) steht, fügt <p>Das ist Textabschnitt!</p> einen Textabschnitt mit ebendiesem Text in das HTML-Dokument ein. Weitere wichtige Tags wären beispielsweise:

  • <a> - Hyperlinks
  • <img> - Grafiken
  • <h1> bis <h6> - Überschriften (desto kleiner die Zahl, desto größer die Überschrift)
  • <strong> - Fettgedruckter Text
  • <i> - Kursiver Text
